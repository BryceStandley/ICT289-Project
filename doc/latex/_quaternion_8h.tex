\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_quaternion}{Quaternion}} functionality for open\+GL. 

Converts a euler angle vector into a quaternion.

Scales a quaternion by a set factor.

Converts a mat4 matrix back into a quaternion.

Converts a rotation vector in radians to degrees / euler angles.

Converts starndard degree angles in a euler vector 3 to quaternion data.

Converts a rotation quaternion to a 4 x 4 matrix.

Creates a new local rotation quaternion with around axis and angle.

Creates a new quaternion with given w, x, y, z components.

Multiply quaternion A by B. NOTE multiplying A by B is not the same as multiplying B by A.

Used to normalise a quaternion between 0 and 1.

Finds the magnatude of a quaternion.

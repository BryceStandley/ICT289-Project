\doxysubsection{Detailed Description}
Used to load and manage models into the program. 

Sets the name of a object.

Updates a game object\textquotesingle{}s information with a temp game object.

Uses open\+GL rotatef to rotate a given axis in degrees.

Uses open\+Gl rotatef to rotate around a give axis in radians.

Translates the scene to a given vec3.

Translates the scene to a gameobjects position.
